Notion → AI Context

Part 2 · Auto-Context from Calls

Voice Recording → Agent Info

When a call recording is processed (Oreka), the AI automatically knows who made the call — agent name, team, targets, performance context — without anyone typing it in.
1 Oreka localParty → matched to agent profile
2 Name, team, today's sales, daily target injected
3 AI coaching becomes personalized per agent
